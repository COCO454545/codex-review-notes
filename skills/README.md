# Personal Skills Mirror

This folder stores lightweight personal skills that are safe to sync across machines.

Rules:

1. Sync only user-created or deeply personalized skills.
2. Do not mirror full third-party repositories unless they have been distilled into a personal skill.
3. Keep large media, generated outputs, caches, dependency folders, secrets, and project data out of this folder.
4. Small reference assets are allowed only when they are essential for the skill, such as a compact style preview or tiny voice style sample.

Current synced skills:

- `ai-video-production-skill`: personal AI video production workflow, including reference learning, preview proof, narration, render verification, cleanup, and skill evolution.
